推荐一款在github的开源项目hexo-markmap。

hexo-markmap

在你的博客中使用markdown插入思维导图，使用markmap。

安装

npm install hexo-markmap

或

yarn add hexo-markmap

使用

{% markmap 300px %}
- Testa
  - test1
  - test2
- Testb
  - test1
  - test2
{%endmarkmap%}
